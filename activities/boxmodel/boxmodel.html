Box Model

Question 1: What is the difference between block tags and inline tags?
Block tags take up the full width available and start on a new line, while inline tags take up only as much width as necessary and do not start on a new line.

Question 2: What is the margin, border, and padding?
Margin is the space outside the border of an element, border is the line surrounding the padding and content, and padding is the space between the content and the border.

Question 3: What are units of measurement you can use in CSS?
Units of measurement in CSS include pixels (px), ems (em), rems (rem), percentages (%), viewport width (vw), viewport height (vh), and more.

Question 4: When would you use a figure tag?
The figure tag is used to group media content (like images, illustrations, diagrams, etc.) along with their captions, making it easier to associate them together.

Margin, Border, and Padding

These are the key components of the CSS box model:

  • Margin: Space outside the border of an element.
  • Border: The line surrounding the padding and content.
  • Padding: Space between the content and the border.

Units of Measurement

CSS supports various units such as:

  • px - Absolute pixels
  • em - Relative to the font size of the element
  • rem - Relative to the root font size
  • % - Percentage relative to the parent element
  • vw - Viewport width
  • vh - Viewport height
