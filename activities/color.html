Colors

Question 1: List out the different ways to select an item.
Items can be selected using tag names, class names, IDs, attribute selectors, pseudo-classes, and pseudo-elements.

Question 2: What is the difference between an ID and a Class?
An ID is a unique identifier for a single element, while a class can be used to identify multiple elements.

Question 3: What is the difference between a SPAN tag and a DIV tag?
A SPAN tag is an inline element used to group inline-elements, while a DIV tag is a block-level element used to group block-level elements.

Question 4: What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?
A DIV tag is a generic container for content with no specific meaning, while HEADER, MAIN, and FOOTER are semantic elements that define specific parts of a webpage.

Question 5: What are the different ways to select a color?
Colors can be selected using color names, HEX codes, RGB values, RGBA values, HSL values, and HSLA values.
