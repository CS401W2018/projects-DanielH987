Structural Tags

Question 1: What goes in the header?
The header usually contains the logo, site title, and navigation links.

Question 2: What goes in the Nav? Can the Nav go in the header? Does it have to go in the header? Does the Nav contain links to another website?
The nav contains navigation links. Yes, it can go in the header, but it doesn't have to. The nav can contain links to other websites but external links should be avoided.

Question 3: What goes in Main?
The main section contains the main content of the page.

Question 4: What goes in the Footer?
The footer usually contains copyright information, contact info (Social links), and links to privacy policies.

Question 5: HTML Validation
HTML validation ensures your HTML code is correct and follows web standards.
