Graphics

Question 1: What kind of photos are good to encode using JPEG?
Photos with many colors and gradients, such as photographs, are good to encode using JPEG.

Question 2: What kind of photos are good to encode using GIF?
Photos with fewer colors, simple graphics, and animations are good to encode using GIF.

Question 3: What is unique about the PNG format?
PNG format supports lossless compression and transparency, making it ideal for images with text, graphics, and transparent backgrounds.

Question 4: How do you resize your photo to the size it should be displayed?
We can resize a photo using photo editing software to the exact dimensions it should be displayed.

Question 5: Why should you resize photos using photo editing software instead of resizing it using CSS?
Resizing photos using photo editing software reduces the file size and improves loading times, whereas resizing using CSS only changes the display size without reducing the file size.
