Internet

The Internet is a global network that connects computers and devices to share information and communicate.

Web

The Web is a part of the Internet where we can access websites, read articles, and watch videos using browsers.

How are the Internet and Web Different?

The Internet is the network, while the Web is a service on that network used for accessing content like websites.

ASCII and Unicode

ASCII is a way to represent text using numbers. Unicode is a newer system that supports many languages and symbols.

What is UTF-8?

UTF-8 is a type of Unicode encoding that stores text efficiently and supports many characters.

URL

A URL (Uniform Resource Locator) is the address used to access a webpage.

Domain Name

A domain name is the readable name for a website, like "example.com".

HTTP

HTTP (Hypertext Transfer Protocol) is a way for browsers and servers to communicate.

HTTPS

HTTPS is like HTTP but more secure because it encrypts the data being sent.

IP Address

An IP address is a unique number that identifies a device on the Internet.

DNS

DNS (Domain Name System) translates domain names into IP addresses.

W3C Recommendations

W3C Recommendations are guidelines for building websites that work well on the Web.

XML, XHTML, HTML, HTML5

These are languages used to structure and present content on the Web. HTML5 is the latest version with new features.
