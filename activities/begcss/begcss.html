CSS

Question 1: Explain the difference between inline, embedded, and external CSS.
Inline CSS is used to apply a unique style to a single HTML element, it uses the style attribute within the element. Embedded CSS is used to define styles in the head section of the HTML document, using the <style> tag. External CSS is used to define styles in an external file, which is linked to the HTML document using the <link> tag.

Question 2: Explain why you should only use external CSS.
External CSS is preferred because it keeps the HTML file clean and separates content from design. It also allows for easier maintenance and reusability of styles across multiple pages.

Question 3: What is a selector used for?
A selector is used in CSS to target HTML elements and apply styles to them. It specifies which elements in the HTML document should be styled.
