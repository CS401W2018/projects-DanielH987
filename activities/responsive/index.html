Responsive

Question 1: Why is the viewport meta tag important?
The viewport meta tag is important because it controls the layout on mobile browsers. Without it, mobile devices will render the page at a desktop width, making it difficult to read and interact with. It ensures that the page is responsive and adapts to different screen sizes.

Question 2: How can you adjust the CSS for printing?
We can adjust the CSS for printing by using a separate stylesheet with media queries for print, or by adding print-specific styles within the main stylesheet using the @media print rule. This allows us to hide unnecessary elements, change font sizes, and adjust layouts for better print formatting.

Question 3: When would you want to adjust the CSS for printing?
We would want to adjust the CSS for printing when we expect users to print the web page. This is common for pages with articles, reports, invoices, or any content that users might need a hard copy of. Proper print styles ensure that the printed version is clean, readable, and well-formatted.
