Table

Question 1: When do you use a table?
Tables should be used to display tabular data, where data is organized in rows and columns.

Question 2: When do you not use a table?
Tables should not be used for layout purposes. Use CSS for layout instead.

Question 3: When do you need to do extra work for table accessibility?
Extra work is needed for table accessibility when the table is complex, such as when it has multiple levels of headers, merged cells, or requires additional context for screen readers.
