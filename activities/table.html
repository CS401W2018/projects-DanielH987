Tables

Question 1: When do you use a table?
Tables should be used to display tabular data, where data is organized in rows and columns.

Question 2: When do you not use a table?
Tables should not be used for layout purposes. We use CSS for layout instead.

Question 3: When do you need to do extra work for table accessibility?
Extra work is needed for table accessibility when the table is complex, such as when it has multiple levels of headers, merged cells, or requires additional context for screen readers.

Question 4: Why don't we use the border attribute?
The border attribute is outdated and should be avoided. Instead, we use CSS to style table borders for better control and separation of content and presentation.

Question 5: What is the easiest way to color every other row a different color?
The easiest way to color every other row a different color is by using the CSS :nth-child pseudo-class. For example: 


            table tr:nth-child(even) {
            background-color: #f2f2f2;
            }
