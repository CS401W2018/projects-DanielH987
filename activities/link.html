Question 2: Explain the difference between an absolute link and a relative link. Why would you use relative links?
An absolute link contains the full URL, including the protocol (e.g., https://), and points to a specific location on the web. A relative link points to somewhere in this website. Relative links are useful for linking to other pages within the same website because they are shorter and more flexible if the domain name changes.

Question 4: List the 3 rules with relative links.
1. Use a leading slash (/) to link from the root directory.
2. Use a double dot (..) to move up one directory level.
3. Use a single dot (.) to link within the same directory.
